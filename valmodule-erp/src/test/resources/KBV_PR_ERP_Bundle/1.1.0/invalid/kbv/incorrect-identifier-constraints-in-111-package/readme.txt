Im 1.1.1. Package von KBV haben einige Constraints die Möglichkeit der URN-Referenzierung nicht berücksichtigt. Diese Beispiele sind inhaltlich nicht valide und hätten eigentlich nicht als solche vom 1.1.1 FHIR-Package erkannt werden müssen. Aufgrund der Nebeneffekte der Constraints werden diese Constrains trotzdem vom HAPI zu false ausgewertet und die Instanzen werden als invalide bewertet.
